Full Stack JavaScript Developer Feb 2016 - Jul 2016

At Code Fellows I took three courses each culminating in a week long effort to design and build a functional product with a small team of class mates. I often found myself in a leadership role or acting as a TA, as my previous experience helped me troubleshoot other student’s code and avoid common beginner mistakes. While many of my CSS and HTML capabilities exceeded what was being covered in these courses, they helped me improve my file organization with MVC principals and introduced mobile first CSS as a best practice.

In the 201 course everything is done with vanilla javaScript, HTML and CSS. It was a great way to break habits of relying on libraries like jQuery for DOM traversal and provided an introduction into object oriented programming in javaScript. For our final project we created a searchable board game library that worked with local storage.

In 301 Code Fellows introduces libraries, frameworks and 3rd party APIs. I already had extensive experience using jQuery, but in my previous jobs I rarely worked with external APIs. For our final project we created a proxy server to work with NOAAs API and used functional programming to filter the data into low and high tide readings for different beaches in Washington State.

In the 401 course we began creating our own APIs and working with single page application frameworks like angular and react for front end development. Primarily we developed express servers, but experimented with a few other node.js server modules. We used mocha to develop unit tests and created an end to end test using protractor. For our final project we developed an Angular application for our express server which interfaced with Philips Hue REST API. Our application included user authentication, multiple views and a mongoDB.

After completing the Full Stack javaScript Development course at Code Fellows, I was offered a position as a 401 Teacher’s Assistant. Usually the 401 position only goes to an existing TA with experience in a 201 or 301 class, but my 401 instructor had personally recommended me. Unfortunately the timing didn’t work out, as my due date was a couple weeks into the 10 week course.
